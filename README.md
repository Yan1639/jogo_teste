# jogo_teste

Projeto de estudo desenvolvido com foco em **aprendizado prático de Git, GitHub e desenvolvimento de jogos 2D**.

Este repositório serve como um laboratório pessoal para experimentar versionamento, organização de código e mecânicas básicas de um jogo de plataforma em Python.

---

## Tecnologias utilizadas
- Python
- Pygame

---

## Objetivo do projeto
- Aprender e praticar:
  - Versionamento com **Git**
  - Uso do **GitHub** (commits, branches e histórico)
  - Organização de um projeto real de software
- Desenvolver a **base de um jogo 2D de plataforma**
- Explorar conceitos como:
  - Loop de jogo
  - Movimentação do personagem
  - Sistema de combate
  - Inimigos e colisões
  - Interface e HUD

---

## Status do projeto
*Em desenvolvimento / Projeto de estudo*

O código está em constante mudança e pode conter partes experimentais ou não finalizadas.
